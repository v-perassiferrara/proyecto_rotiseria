# Proyecto Rotisería UM GANG
---

### Descripción
Este es el proyecto anual para la materia de **Programación I**. La aplicación consiste en una rotisería online, permitiendo a los usuarios registrarse, iniciar sesión, ver productos, realizar pedidos y más. Cuenta con un panel de administración para gestionar usuarios, productos, y pedidos.

### Instalación y Uso

**Backend:**
1. Navegar al directorio `backend`.
2. Ejecutar el script de instalación para crear el entorno virtual e instalar dependencias:
   ```bash
   $ ./install.sh
   ```
3. Ejecutar la aplicación:
   ```bash
   $ ./boot.sh
   ```

**Frontend:**
1. Navegar al directorio `frontend/app_rotiseria`.
2. Instalar las dependencias del proyecto:
   ```bash
   $ npm install
   ```
3. Iniciar el servidor de desarrollo de Angular:
   ```bash
   $ ng serve
   ```
4. Abrir el navegador en `http://localhost:4200/`.
